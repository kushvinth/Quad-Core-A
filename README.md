#  AI Health Risk Advisor

An AI-powered system that **assesses symptom urgency (not diagnosis)** and guides users to the **right medical action**, reducing unnecessary hospital visits and ensuring timely emergency care.

---

##  Features

* Text & voice symptom input
* Risk-level assessment
* Emergency escalation
* Nearby hospital recommendations
* First-aid & next-step guidance

---

##  Tech Stack

Gemini API 路 Firestore 路 Google Maps API 路 Speech-to-Text 路 BigQuery 路 Firebase Hosting
